 Dashboard BI — Analyse des Ventes E-Commerce 2024
 Description

Ce projet est un dashboard interactif en HTML/CSS/JavaScript permettant de visualiser les performances d’un système e-commerce en 2024.

Il présente une interface moderne de type Business Intelligence (BI) avec des indicateurs clés (KPI), graphiques dynamiques et tableau de performance.
 Aperçu du projet

Le dashboard permet de suivre :

📈 Évolution des revenus mensuels (2023 vs 2024)
🛒 Nombre de commandes
💰 Chiffre d’affaires
👥 Clients actifs
🔁 Taux de retour
📦 Top produits par performance
📊 Analyse par catégorie et conversion
